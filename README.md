# Login Page using HTML, CSS, and JavaScript 🔐

A modern, responsive login form built with pure HTML, CSS, and JavaScript. This project provides a secure user authentication interface with real-time form validation, smooth animations, and a clean design. Perfect for integrating into web applications requiring user login functionality. 📧🔒

## 🚀 Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices 📱
- **Real-time Validation**: Instant email format checking and password presence validation ✅
- **Visual Feedback**: Shake animation on invalid inputs and color-coded validation states 🎭
- **Font Awesome Icons**: Integrated icons for email, password, and error states 🎨
- **Modern UI**: Clean, professional design with Google Fonts (Poppins) and smooth transitions ✨
- **Accessibility**: Proper form labels, focus states, and semantic HTML ♿
- **No Dependencies**: Pure vanilla JavaScript, no external libraries required (except Font Awesome CDN) 🚀

## 🛠️ Technologies Used

- **HTML5**: Semantic markup and form structure
- **CSS3**: Modern styling with animations, flexbox, and responsive design
- **JavaScript (ES6+)**: Form validation logic and DOM manipulation
- **Font Awesome**: Icon library for visual elements
- **Google Fonts**: Poppins font family for typography

## 📦 Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/login-page.git
   cd login-page
   ```

2. **Open in browser**:
   Simply open `index.html` in your preferred web browser. No server required! 🌐

## 🎯 Usage

1. Open `index.html` in a web browser
2. Enter a valid email address in the email field
3. Enter your password in the password field
4. Click "Login" to submit the form
5. The form validates inputs in real-time and provides feedback

### Validation Rules
- **Email**: Must match standard email format (e.g., user@example.com)
- **Password**: Cannot be empty
